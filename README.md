# Startup 3 - Figma Landing Page

A responsive multi-section landing page built using **HTML5** and **CSS3**, recreated from a Figma design as a frontend practice project.

---

## Preview

This project recreates a modern startup landing page featuring:

- Hero Section
- Content Section
- Signup Section
- Testimonials
- Portfolio / Showcase
- Pricing Plans
- Footer

---

## Technologies Used

- HTML5
- CSS3
- Google Fonts (DM Sans)
- Font Awesome Icons


```bash

```


